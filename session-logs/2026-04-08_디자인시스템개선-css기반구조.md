## 3. P0: **faneasy** 메인페이지 개선

### 변경사항

#### 3.1 CSS 변수 토큰 시스템 추가

```css
/* Spacing Tokens */
--spacing-xs: 4px
--spacing-sm: 8px
--spacing-md: 16px
--spacing-lg: 24px
--spacing-xl: 40px
--spacing-2xl: 64px
--spacing-3xl: 100px

/* Typography Tokens */
--letter-spacing-tight: -0.035em    /* 제목 */
--letter-spacing-normal: -0.01em    /* 버튼 */
--letter-spacing-relaxed: 0.01em    /* 작은 텍스트 */

/* Transitions */
--transition-fast: 100ms
--transition-base: 200ms
--transition-slow: 300ms
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1)
```

#### 3.2 다크모드 지원 추가

**라이트 모드 (기본)**:
```css
--background: #ffffff
--text-primary: #0f0f0f
--accent: #5b5fef
```

**다크 모드** (`[data-theme="dark"]`):
```css
--background: #0f0f0f
--text-primary: #f5f5f5
--accent: #6b70ff (라이트 조정)
```

**부드러운 전환**:
```css
@media (prefers-reduced-motion: no-preference) {
  * {
    transition: background-color 200ms,
                color 200ms,
                border-color 200ms;
  }
}
```

#### 3.3 자간 정의 추가

```css
h1, h2, h3, h4, h5, h6 {
  letter-spacing: -0.035em;  /* 제목은 단단하게 */
}

.btn {
  letter-spacing: -0.01em;   /* 버튼은 약간 단단하게 */
}

a {
  letter-spacing: 0.005em;   /* 링크는 약간 풀어서 */
}
```

#### 3.4 컴포넌트 시스템 강화

**버튼 variants**:
- `.btn-primary`: 검은 배경, 흰 텍스트
- `.btn-secondary`: 투명 배경, 테두리
- `.btn-accent`: 보라색 배경 (#5b5fef)
- `.btn-ghost`: 투명, 텍스트만

**카드 시스템**:
```css
.card {
  background: var(--surface-primary);
  border: 1px solid var(--border-light);
  border-radius: 12px;
  padding: 24px;
  transition: all 200ms;
}

.card:hover {
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.12);
  border-color: var(--border-medium);
}
```

#### 3.5 접근성 강화

**Selection 스타일** (Ctrl+A로 드래그 시 보라색):
```css
::selection {
  background: #5b5fef;
  color: #ffffff;
}

::-moz-selection {  /* Firefox */
  background: #5b5fef;
  color: #ffffff;
}
```

**포커스 상태**:
```css
a:focus-visible {
  outline: 2px dashed #5b5fef;
  outline-offset: 4px;
}

button:focus-visible {
  outline: 2px solid #5b5fef;
  outline-offset: 2px;
}
```

#### 3.6 타이포그래피 시스템

```css
h1 { font-size: clamp(2rem, 6vw, 4.5rem); }
h2 { font-size: clamp(1.75rem, 4vw, 3.5rem); }
h3 { font-size: clamp(1.25rem, 3vw, 2rem); }

p {
  font-size: 1rem;
  line-height: 1.7;
  color: var(--text-secondary);
}
```

#### 3.7 모바일 최적화

```css
@media (max-width: 768px) {
  section { padding: 100px 0; }
  h1 { font-size: 2rem; }
  h2 { font-size: 1.5rem; }
}
```

### 파일 통계

```css

  --text-primary: #f5f5f5;
  --text-secondary: #b0b0b0;
  --text-muted: #808080;
  
  --accent-purple: #c084fc;  /* 밝게 조정 */

/* 부드러운 전환 */

  * {
    transition: background-color 200ms,
                color 200ms,
                border-color 200ms;
  }
}
```

#### 4.3 애니메이션 시스템 추가

```css

  from { opacity: 0; }
  to { opacity: 1; }
}

  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }

.fade-in { animation: fadeIn 200ms ease-in-out; }
.slide-up { animation: slideUp 300ms ease-in-out; }
```

## 8. 기술적 포인트

### CSS 변수의 이점
1. **일관성**: 전체 프로젝트에서 동일한 값
2. **유지보수**: 한 곳에서 수정 → 모든 곳 반영
3. **다크모드**: selector 변경만으로 모든 변수 자동 전환
4. **성능**: CSS 레벨에서 처리 (JavaScript 불필요)

### @layer 구조
```css
@layer base {      /* 리셋, 폰트, 변수 */
  /* html, body, 기본 요소 */
}

@layer components { /* 버튼, 카드 등 */
  /* 재사용 가능한 클래스 */
}

@layer utilities {  /* 헬퍼, 애니메이션 */
  /* 추가 유틸리티 */
}
```

### 접근성 베스트 프랙티스
- `prefers-reduced-motion` 존중
- `focus-visible` 명확한 스타일
- `::selection` 색상 대비 확보
- 모든 컬러: WCAG AA 이상
