# matecheck-web (메이트체크 웹)

## 공통 지침
- 빌더 공통 지침: 맥미니 루트 `gstack.md` 참조
- Rails API: `dev/rails/matecheck/` (포트 4010)

## 프로젝트 개요
- **서비스명**: 메이트체크
- **설명**: 공유 주거(쉐어하우스) 미션/가계부/일정/규칙 통합 관리 플랫폼
- **기술 스택**: Next.js 16, TypeScript, Tailwind CSS
- **배포**: Vercel (vibers-leo/matecheck 레포)
- **백엔드**: Rails API on NCP Docker (port 4010)

## 세션로그 기록 (필수)
- 모든 개발 대화의 주요 내용을 `session-logs/` 폴더에 기록할 것
- 파일명: `YYYY-MM-DD_한글제목.md` / 내용: 한글
- 세션 종료 시, 마일스톤 달성 시, **컨텍스트 압축 전**에 반드시 저장
